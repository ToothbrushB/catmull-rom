About This App

This web application allows users to upload a background image, such as a floor plan, and define a smooth path using Catmull-Rom splines by clicking to add control points on the canvas. It is particularly useful for robotics and path planning scenarios where smooth motion along a predefined track is needed.

Features include:

  • Upload a background image (e.g., a map or room layout).

  • Add random shapes for testing or decoration.
